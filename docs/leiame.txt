// Instruções de Execução

Para executar o arquivo somamax.c, é necessário que seja instalado na máquina o compilador GCC.
Executar o terminal na pasta onde se encontra o arquivo e digitar o seguinte comando:
    >> gcc -o somamax somamax.c

Será gerado um arquivo executável, que deve então ser iniciado a fim de utilizar o programa. 
Comando a ser executado no terminal:
    >> somamax


O mesmo deve ser feito com o qmagico.c, porém com o comando
    >> gcc -o qmagico qmagico.c
E para executar
    >> qmagico
